Newman -Commandline tool
Execute the Postman collections from the command line


Precondition:
==============

1) Export collection file and variable file to folder
2) Place the data files if data  is read from extrenal source


Combinations for execution:
===========================

1)Collection with variables without report

newman run Week2Day1.postman_collection.json -e QA.postman_environment.json -g workspace.postman_globals.json

2)Collection with variables generating report

newman run Week2Day1.postman_collection.json -e QA.postman_environment.json -g workspace.postman_globals.json -r htmlextra

3)Collection without variables without report

4)Collection without  variables generating report

5)With external data file


newman run Week2Day1.postman_collection.json -e QA.postman_environment.json -g workspace.postman_globals.json -d JsonData.json -r htmlextra


6) How to specify the number of iterations

newman run Week2Day1.postman_collection.json -e QA.postman_environment.json -g workspace.postman_globals.json -d JsonData.json -r htmlextra -n 5

============
